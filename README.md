# mlService_baseapp

Based on [ngX Starter Kit](https://github.com/ngx-rocket/starter-kit). Check it out for further information (e.g. building).

Web project starter kit including modern tools and workflow based on
[angular-cli](https://github.com/angular/angular-cli), best practices from the community, a scalable base template and
a good learning base.

Generated using [ngX-Rocket](https://github.com/ngx-rocket/generator-ngx-rocket).

# Getting started

1. Go to project folder and install dependencies:
 ```bash
 npm install
 ```
 
2. Launch development server, and open `localhost:4200` in your browser:
 ```bash
 npm start
 ```

## Development server

Run `npm start` for a dev server. Navigate to `http://localhost:4200/`. The app will automatically reload if you change
any of the source files.
You should not use `ng serve` directly, as it does not use the backend proxy configuration by default.

## Generate a new view

Run `npm run generate -- component <name>` and `npm run generate -- module <name>` to generate a new component.
